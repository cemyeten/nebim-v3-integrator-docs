# Nebim V3 Integrator Dokümantasyonu

Bu repo, Nebim V3 Integrator Web Servisi için hazırlanmış Türkçe teknik dokümantasyonu içermektedir.

Dokümantasyon kapsamında aşağıdaki modeller detaylı olarak açıklanmıştır:

- Connect (Bağlantı Modeli)
- Sorgu Çalıştırma (RunQuery)
- Procedure Çalıştırma (RunProc)
- Müşteri, Ürün gibi kart tanım oluşturma modelleri
- Sipariş, Fatura gibi fiş hareketi oluşturma modelleri


---

## Nebim V3 Integrator Nedir?

Nebim V3 Integrator;

- IIS üzerinde çalışan
- ASP.NET tabanlı
- RESTful mimariye sahip
- JSON Request / JSON Response yapısında çalışan

bir Web API uygulamasıdır.
